# Personalization Guide

Use this guide when a prompt was written for a specific person and you want to adapt it for yourself or someone else.

## Replace These First
- Name, location, and current role
- Years of experience and proof signals
- Core strengths, niche, and tools
- Audience and career direction
- Tone and voice instructions
- Offer, positioning, or service lane references
- Any portfolio, README, case study, or LinkedIn references tied to the original person

## Recommended Workflow
1. Gather the new person's LinkedIn summary, resume, README, project list, or short bio.
2. Replace the identity block before changing the style block.
3. Update niche-specific terms, tools, and workflow references.
4. Remove any leftover names, achievements, or examples from the original person.
5. Run one realistic test input.
6. Save the adapted version as a new file.

## Good Practice
- Keep the original master untouched.
- Create one personalized version per person or niche.
- Name the new file clearly, such as `linkedin-repost-v6-jane-doe.md`.
