# Critique and Evaluation

## Module Summary
This folder contains review-oriented prompts used to score, critique, and tighten outputs after generation. Use these when you want a stronger QA pass before publishing or reusing content.

## Directory Structure
```text
└── critique-prime-evaluator.md  # prompt file
```

## How To Use This Folder
- Use this after generation, not before.
- Pair it with any content, README, or LinkedIn prompt as a second-pass QA step.

## Workflow Pattern
1. Pick the closest prompt instead of starting from a blank page.
2. Copy the prompt block from the chosen file.
3. Supply the source material, constraints, and target outcome.
4. Save improved variants as new versions when behavior changes materially.

## Maintenance Notes
- Keep names short, searchable, and folder-specific.
- Add new versions only when the output behavior changes clearly.
- Prefer updating this README when new files are added.
